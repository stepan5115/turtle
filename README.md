# Turtle

Программа для исполнения черепашьих скриптов с визуализацией и поддержкой автотестов.

## Сборка

Для сборки исполняемого файла `turtle` используйте команду:

```bash
make all
```

## Запуск программы

Синтаксис запуска:
```bash
./turtle <файл_с_программой> [--testing]
```
<файл_с_программой> — путь к файлу со скриптом на языке Turtle.

--testing — опциональный параметр, который отключает графический интерфейс и позволяет программе сразу завершаться после выполнения всех команд (используется для автотестов).

Пример:
```bash
./turtle tests/valid/spiral.turtle
./turtle tests/valid/spiral.turtle --testing
```

## Автотесты

Для запуска набора автотестов выполните:
```bash
./testing.sh
```
Скрипт проверяет корректность работы программы на валидных и невалидных тестах и выводит результаты в цвете.

* 🟢Зелёный — тест пройден
* 🔴Красный — тест не пройден

В конце выводится результат (сколько тестов пройдено, сколько провалено)
